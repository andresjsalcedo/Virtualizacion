El editor de Canva tiene varias secciones: la navegación principal, el panel lateral, la barra de herramientas, el lienzo y los controles de página. Para enfocarte en una sección y conocer más sobre cada una, usa el tabulador. Para ir al lienzo, presiona “⌘F2”.
La navegación principal contiene botones con opciones para trabajar sobre el documento. Para pasar de un botón a otro, usa las teclas de flechas a la izquierda y a la derecha y, para activar un botón, usa la tecla Enter. Para ir al lienzo, presiona “⌘F2”.

El título del diseño es el nombre de tu diseño. Para editarlo, presiona la tecla Intro. Para dejar de editar el título del diseño, presiona la tecla Escape.
La barra de herramientas contiene botones para editar el elemento seleccionado. Para pasar de un botón a otro, usa las teclas de flechas a la izquierda y a la derecha y, para activar un botón, usa la tecla Enter. Para ir al lienzo, presiona “⌘F2”.
El lienzo es la página donde creas tu diseño. Para ir allí, presiona la tecla “Entrar”. Para ir a grupos o elementos en la página, vuelve a presionar la tecla “Entrar”. Presiona el tabulador para ir a la siguiente página, al siguiente grupo o al siguiente elemento. Para usar la barra de herramientas, usa la tecla “⌘F1”. Si ya seleccionaste un elemento, podrás moverlo usando las teclas de flechas, rotarlo con la teclas “,” y “.”, o cambiarle el tamaño con las teclas de flechas y la tecla “⌘”. También podrás aumentar los incrementos de estos atajos con la tecla “⇧”.
